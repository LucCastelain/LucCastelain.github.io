This prototype is an advanced crafting system made with Godot engine. It is based on Aterlier Ryza's crafting system.
This crafting system use custom resources to make a really complete crafting system. Each item needs one or more specific types, subtypes or unique ingredients to be crafted. The Crafted item takes in account the quality of each ingredients used.

It is easy to add new items for craft and modify custom resourses to extant the depth of this crafting system.
It can be easily extanded in multiple ways. For example crafted items could take the specific characteristic(s) of choosen ingredients...

This prototype is standalone and was thought so it can be easily implemented in a more complex games.
